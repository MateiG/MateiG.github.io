  • Blind Pilot
    Blind Pilot is a mobile app that uses a phone's rear camera to help visually-impaired people navigate real-world obstacles. It works by running a real-time semantic segmentation and path planning algorithm on-device and directs the user with haptic feedback. See more at https://github.com/MateiG/blind-pilot and watch the full demo here.
